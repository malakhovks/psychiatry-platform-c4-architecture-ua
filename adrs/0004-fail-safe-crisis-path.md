# 4. Кризовий маршрут є fail-safe і незалежним від генеративного ШІ

Дата: 2026-06-20

## Статус

Прийнято

## Контекст

Відмова LLM, зовнішнього постачальника або звичайного каналу не повинна блокувати допомогу за критичного ризику.

## Рішення

Виявлення виконує детермінований рушій; ескалація створюється атомарно; доставка потребує підтвердження; після тайм-ауту активується резервний канал. Пацієнтський кризовий елемент доступний окремо від діалогу.

## Наслідки

Потрібні локальні кризові політики, SLA, чергові ролі, резервні канали, навчання й моніторинг end-to-end.
