# 9. Мультимодальні вимірювання проходять обов'язковий quality gate

Дата: 2026-06-20

## Статус

Прийнято

## Контекст

ЕКГ, голос, відео й зображення можуть бути технічно непридатними; автоматичний висновок із поганого сигналу створює клінічний ризик.

## Рішення

До спеціалізованого аналізатора допускається лише потік, який пройшов профіль якості. Результат зберігає якість, невизначеність, версію конвеєра та статус клінічного підтвердження.

## Наслідки

Потрібні окремі критерії для кожної модальності, UX повтору вимірювання й контроль дрейфу аналізаторів.
